solid: 定义实线边框

07.选择器

07.01 元素选择器

CSS 元素/标签选择器通常是某个 HTML 元素， 比如 p、h1、a div 等

07.02 ID选择器

id 选择器可以为标有特定 id 的 HTML 元素指定特定的样式。 id 选择器以 “#” 来定义 。

07.02 class选择器

class 类选择器， 可以通过 class 属性选择去使用这个样式 .

基本语法 :

​ .class 属性值 {属性: 值;}

07.03 组合选择器

组合选择器可以让多个选择器共用同一个 css 样式代码 .

优先级说明 :

​ 行内样式 > ID 选择器 > class 选择器 > 元素选择器

(3).Docker命令

1.Docker命令

1.1 docker search

docker search：在Docker Hub中搜索镜像

1.2 docker pull/push

docker pull：拉取镜像

docker push：上传镜像

1.3 docker images

docker images：列出镜像

1.4 docker save/load

docker save：将一个或多个镜像保存到tar存档(默认情况下流式传输到STDOUT)

docker load：从tar存档或STDIN加载镜像

1.5 docker ps

docker ps：列出容器

1.6 docker run

docker run：在新容器中运行命令

1.7 docker commit

docker commit：根据容器的更改创建新镜像

1.8 docker cp

docker cp：在容器和本地文件系统之间复制文件/文件夹

1.9 docker rm/rmi

docker rm：删除一个或多个容器

docker rmi：删除一个或多个镜像

1.10 docker start/stop/restart

docker start：启动一个或多个已停止的容器

docker stop：停止一个或多个运行中的容器

docker restart：重新启动一个或多个容器

1.11 docker exec

docker exec：在正在运行的容器中运行命令

1.12 docker port

docker port：列出端口映射或容器的特定映射

1.13 docker rename

docker rename：重命名容器

1.14 docker update

docker update：更新一个或多个容器的配置

1.15 docker volume

docker volume：管理卷

  • docker volume create
  • docker volume inspect
  • docker volume ls
  • docker volume prune
  • docker volume rm

1.16 docker tag

docker tag：创建一个引用了SOURCE_IMAGE的标签TARGET_IMAGE

1.17 docker inspect

docker inspect：返回有关Docker对象的低级信息

1.18 docker network

docker network：管理网络

1.19 docker logs

docker logs：获取容器的日志

1.20 docker build

docker build：从Dockerfile构建镜像

1.21 docker login/logout

docker login：登录仓库

docker logout：登出仓库

1.22 docker stats

docker stats：显示容器资源使用情况统计信息的实时流

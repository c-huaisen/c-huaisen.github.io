1.Zabbix简介

Zabbix 是一个企业级的开源分布式监控解决方案。

Zabbix 是一款监控网络的众多参数以及服务器、虚拟机、应用程序、服务、数据库、网站、云等的健康和完整性的软件。 Zabbix 使用灵活的通知机制，允许用户为几乎任何事件配置基于电子邮件的告警，以实现对服务器问题做出快速反应。 Zabbix 基于存储的数据提供出色的报告和数据可视化功能。这使得 Zabbix 成为容量规划的理想选择。

1.2 监控流程

  • 数据采集：通过SNMP、Agent、ICMP、SSH、IPMI等对系统进行数据采集。

  • 数据存储：各类数据库服务，MySQL、PostgreSQL等。

  • 数据分析：提供图形及时间线情况信息，方便我们定位故障所在

  • 数据展示：指标信息、指标趋势展示。

  • 报警机制：电话、邮件、微信、短信、报警升级机制

  • 报警处理：故障级别判定，找响应人员进行快速处理

1.3 监控指标

硬件监控

  • 机器硬件:CPU温度、物理磁盘、虚拟磁盘、主板温度、磁盘阵列

系统监控

  • 主机存活
  • CPU、内存、硬盘、使用率
  • inode
  • 负载
  • 网卡出入带宽
  • TCP连接数
  • 磁盘读写、只读

应用监控

  • MySQL
  • Redis
  • Nginx

网络监控

  • 网络质量
  • 公网出口
  • 专线带宽
  • 网络设备

流量分析

日志监控

安全监控

性能监控(APM)java|php|go|nodejs|分布式链路追踪

  • PinPoint

业务监控

电商业务为例：

  • 每分钟产生多少订单
  • 每分钟注册多少用户
  • 每分钟多少活跃用户
  • 每天有多少推广活动
  • 推广活动引入多少用户
  • 推广活动引入多少流量
  • 推广活动引入多少利润

其他
